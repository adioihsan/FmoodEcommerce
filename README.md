## FmoodEcommerce - E-Commerce Website
### Features :  
#### Guest :
1. Register
2. Login
3. View Product
4. Find Product

#### User :
1. Add to cart
2. Checkout
3. Trace Transaction
4. View Transaction
5. Open Their Own Store

#### Admin Store
1. Product CRUD (Create Read Update Delete)
2. Add a product video and photo upto 5 photo / product
3. Manage Order  
    - Cancel Order
    - Process Order
    - Trace Order
    - View Success Order
